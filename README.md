# Clon-Proyecto-Desarrollo-Web

Sitio web frontend para organizar torneos de tenis con HTML, JavaScript, Bootstrap y `localStorage`.

## Funcionalidades

- Crear, editar y eliminar torneos.
- Registrar, editar y eliminar jugadores.
- Separar jugadores por sexo y nivel de juego.
- Generar brackets automáticamente por categoría.
- Asignar BYE cuando hay participantes impares.
- Capturar puntajes y avanzar ganadores automáticamente.
- Calcular ranking con victorias, derrotas, diferencia de puntos, BYE y campeonatos.
- Persistir datos en `localStorage` del navegador.
- Exportar el esquema y datos del torneo a PDF.
- Imprimir la vista del bracket.

## Tecnologías

- HTML5
- JavaScript
- Bootstrap 5
- CSS personalizado
- localStorage
- jsPDF
- html2canvas

## Uso

Abre `index.html` directamente en el navegador o publica el repositorio con GitHub Pages.

1. Crea un torneo desde la sección **Torneos**.
2. Registra jugadores en la sección **Jugadores**.
3. Genera los emparejamientos desde **Brackets y resultados**.
4. Edita puntajes para avanzar ganadores.
5. Consulta el **Ranking**.
6. Exporta el torneo a PDF desde el botón **Exportar PDF**.
